# Mahiron 5

![logo](https://avatars.githubusercontent.com/u/127649250)

Yet another DVR Tuner Server for Japanese TV.

## 特徴

- Mirakurun互換のAPI
- Go言語で書かれていてシングルバイナリで動作
- 分かりやすいダッシュボード
- リアルタイム番組表更新
- ケーブルテレビの再送信など複数経路のTSを統合
- OpenTelemetryによる高い可観測性

## Mirakurun APIとの差分

- 非対応
  - `/api/config` 関連
